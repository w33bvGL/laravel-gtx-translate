CrawlAnimeFilters.php : 84

Общий метод для извлечения данных фильтра по индексу строки.

private crawlFilterData(int $rowIndex) : JsonResponse

A General Method for Filter Data Extraction by Row Index.

Этот метод делает HTTP запрос для получения HTML-контента страницы, затем использует библиотеку DomCrawler для извлечения данных из таблицы фильтров. Парсит фильтры на основе переданного индекса строки и возвращает их в формате JSON.

This method makes an HTTP request to fetch the HTML content of the page, then uses the DomCrawler library to extract filter data from the table. It parses the filters based on the provided row index and returns them in JSON format.

Parameters
$rowIndex : int

The row index in the table to parse. For example, 0 is "Type", 2 is "Status".

Return values
JsonResponse —

Возвращает распарсенные опции в формате JSON.
