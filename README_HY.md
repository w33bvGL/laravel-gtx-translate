# Google-Translate-Scraper

**Google-Translate-Scraper**-ը PHP Laravel գրադարան է, որը նախատեսված է Google Translate-ը քերելու համար՝ տարբեր լեզուներով տեքստը թարգմանելու համար: Այն ապահովում է պարզ ինտերֆեյս թարգմանություններ կատարելու համար:

Այս փաթեթը թույլ է տալիս թարգմանել տեքստը մի լեզվից մյուսը` քերելով Google Translate-ը: Այն իրականացնում է այնպիսի առաջադրանքներ, ինչպիսիք են հարցումների ժամկետների սահմանումը, վստահված անձերը և բազմաթիվ լեզուների աջակցությունը: Սա լուծում է մշակողների համար, ովքեր ցանկանում են ինքնուրույն թարգմանության գործիք՝ առանց API ստեղների համար վճարելու կամ երրորդ կողմի ծառայություններին ապավինելու:

## Առանձնահատկություններ
- **Google Translate Scraping**. թարգմանեք տեքստը՝ քերելով Google Translate-ի թաքնված API-ը:
- **Պատվիրված ժամկետների**. Սահմանեք նվազագույն և առավելագույն ժամկետները հետաձգելու հարցումները:
- **Վստահված անձի աջակցություն**. Օգտագործեք վստահված անձ քերելու համար, որպեսզի խուսափեք տոկոսադրույքների սահմանափակումներից և արգելափակումից:
- **Աջակցվող լեզուներ**. Հեշտությամբ թարգմանեք Google Translate-ի կողմից աջակցվող բազմաթիվ լեզուների միջև:
- **Կարգավորելի օգտագործողի գործակալներ**. ընդօրինակեք տարբեր բրաուզերներ` հայտնաբերումից խուսափելու համար:

## Տեղադրում
Դուք կարող եք տեղադրել **google-translate-scraper** գրադարանը Composer-ի միջոցով: Դա անելու համար գործարկեք հետևյալ հրամանը ձեր տերմինալում.

```bash
կոմպոզիտորը պահանջում է anidzen/google-translate-scraper
```

## Կազմաձևում
Գրադարանը տեղադրելուց հետո դուք պետք է կարգավորեք դրա պարամետրերը «google-translate-scraper.php» կոնֆիգուրացիայի ֆայլում: Ֆայլում կարող եք նշել հետևյալ պարամետրերը.

- **base_url**. URL հիմնական ռեսուրսի համար:
- **hidden_api_base_url**՝ URL թաքնված Google Translate API-ի համար:
- **timeout_min** և **timeout_max**. Սահմանեք հարցումների նվազագույն և առավելագույն ժամանակը:
- ** վստահված անձ**. անհրաժեշտության դեպքում ստեղծեք վստահված անձ:
- **text_max_length**. տեքստի առավելագույն երկարությունը, որը կարելի է թարգմանել:
- **supported_languages**. Աջակցվող լեզուների ցանկ:
- **user_agents**. Անհատականացված User-Agent տարբեր բրաուզերների մոդելավորման համար:

## Օգտագործման օրինակ

Տեքստը թարգմանելու համար օգտագործեք հետևյալ օրինակը.

```php
<?php

Անվանատարածք Anidzen\GoogleTranslateScraper;

օգտագործել Anidzen\GoogleTranslateScraper\Facades\TextTranslator;
օգտագործել Բացառություն;

դասի GoogleTranslator
{
 մասնավոր տող $sourceLanguage;
 մասնավոր տող $targetLanguage;

 հանրային ֆունկցիա __construct (string $sourceLanguage = 'en', string $targetLanguage = 'ru')
 {
 $this->sourceLanguage = $sourceLanguage;
 $this->targetLanguage = $targetLanguage;
 }

 /**
 * Թարգմանում է տեքստը մի լեզվից մյուսը:
 *
 * @param string $text Տեքստ թարգմանելու համար:
 * @return string Թարգմանված տեքստ:
 * @throws Բացառություն Եթե թարգմանության ժամանակ սխալ է տեղի ունեցել:
 */
 հանրային ֆունկցիա translateText(string $text): string
 {
 փորձիր {
 $response = TextTranslator::translate ($this->sourceLanguage, $this->targetLanguage, $text);

 վերադարձնել $response->getContent();
 } catch (բացառություն $e) {
    նետում նոր Բացառություն('Թարգմանությունը ձախողվեց. ' . $e->getMessage());
 }
 }
}
```
